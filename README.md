# SkipRec

## Requirements

- Python 2.7
- TensorFlow 1.15.0
- NumPy 1.16.6
- Pandas 0.24.2


## Quick Starts

Please put dataset in `/data` or use `--data_folder` to specify.

### NextItNet

- Original 

```shell script
sh run-nin-ori.sh
```

- NextItNet+ 

```shell script
sh run-nin-plus.sh
```

- SkipRec-Gumbel

```shell script
sh run-nin-gumbel.sh
```

- SkipRec-RL

```shell script
sh run-nin-rl.sh
```

- SkipRec-Gumbel-GRU

```shell script
sh run-nin-gru-gumbel.sh
```

- SkipRec-RL-GRU

```shell script
sh run-nin-gru-rl.sh
```

### SASRec

- Original

```shell script
sh run-sas-ori.sh
```

- SkipRec-Gumbel

```shell script
sh run-sas-gumbel.sh
```

- SkipRec-RL

```shell script
sh run-sas-rl.sh
```
